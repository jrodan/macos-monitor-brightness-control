# Contributing

Thank you for your interest in contributing!

## Project Status
This is a personal project intended to be a simple, lightweight utility. I (the author) do not intend to provide active support or significant feature developments.

## How to Contribute
- **Bug Fixes**: Pull requests for fixes are welcome.
- **Features**: Please open an issue before starting work on a major feature.
- **Forks**: You are free to fork the repository for your own use, as long as it remains non-commercial.

## Code Style
Please maintain the existing Swift/SwiftUI patterns used in the project.

## Commit Messages
This project follows [Conventional Commits](https://www.conventionalcommits.org/en/v1.0.0/).
- `feat:` for new features
- `fix:` for bug fixes
- `chore:` for maintenance
- `docs:` for documentation updates
